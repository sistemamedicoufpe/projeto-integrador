# projeto-integrador
Funcionalidades Implementadas:
1. Cadastro de Pacientes
Modal completo para cadastro de novos pacientes

- Validação de campos obrigatórios

- Cálculo automático da idade

- Geração automática de ID do paciente

2. Visualização de Pacientes Ativos
Tabela com lista completa de pacientes

- Filtragem por médico logado

- Indicadores de risco

- Ações para visualizar e editar pacientes

3. Gestão de Testes Realizados
Lista de todos os testes concluídos

- Filtragem por paciente

- Detalhes completos de cada teste

- Histórico de aplicação

4. Sistema de Diagnósticos
Registro de diagnósticos realizados

- Classificação por gravidade

- Relacionamento com testes aplicados

- Histórico completo

5. Relatórios e Impressão
Múltiplos tipos de relatórios

- Geração de PDF

- Funcionalidade de impressão

- Relatórios específicos por categoria

6. Navegação Melhorada
Sistema de views para diferentes seções

- Navegação lateral intuitiva

- Estado da aplicação persistente

7. TypeScript
- Interfaces bem definidas para todos os tipos de dados

- Tipagem forte para melhor desenvolvimento

- Estrutura escalável para novas funcionalidades
